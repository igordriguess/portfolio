PowerShell

O PowerShell é uma poderosa ferramenta de automação e scripting, que utilizo para simplificar a administração de ambientes e executar comandos de forma remota. Com ele, otimizo processos, automatizo tarefas repetitivas e melhoro a eficiência na gestão de sistemas, garantindo agilidade e precisão na execução das atividades.

Python

Python é uma linguagem de programação versátil e poderosa, com suporte a paradigmas como orientação a objetos, programação funcional e tipagem dinâmica. Utilizo o Python no desenvolvimento de automações e aplicativos para gestão de ambientes, empregando o Visual Studio Code como IDE, o que me permite criar soluções robustas e eficientes que otimizam processos e melhoram a administração de sistemas.

Azure

O Azure é a plataforma de nuvem da Microsoft, oferecendo uma ampla gama de serviços para empresas e desenvolvedores. No meu dia a dia, utilizo o Azure para gerenciar recursos de forma eficiente, configurar rotinas de segurança, realizar análises de dados e garantir a integridade das informações por meio de processos de backup e restauração. Essa experiência me permite otimizar operações na nuvem, mantendo um ambiente seguro e bem estruturado.

OCI

O Oracle Cloud Infrastructure (OCI) é a plataforma de nuvem da Oracle, que oferece uma vasta gama de serviços, incluindo infraestrutura, banco de dados e ferramentas para desenvolvedores. Minha atuação diária no OCI envolve a gestão e configuração de novas máquinas virtuais (VMs), ajuste de recursos alocados, configuração de Load Balancers, Network Security Groups (NSG) e Web Application Firewall (WAF), além de outras rotinas essenciais para garantir a eficiência, segurança e escalabilidade do ambiente em nuvem.
